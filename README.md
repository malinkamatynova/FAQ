# Горячие клавиши
### Копировать:
```bash
 ctrl+c
 ```
### Вставить:
```bash
ctrl+v
```
### Сохранить:
```bash
ctrl+s
```
### Вырезать:
```bash
ctrl+x
```
### Отмена:
```bash
ctrl+z
```
# Работа с git
### Скачать проект:
```bash
git clone <URL>
``` 
### Учесть все файлы:
```bash
git add .
```
### Сохранить коммит:
```bash
git commit -m "..."
```
### Отправка в облако:
```bash
git push
```
